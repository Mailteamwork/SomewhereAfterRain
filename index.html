คู่มือการใช้งานแอปพลิเคชัน

1. การเพิ่มทริปแผนใหม่ (➕)

แตะปุ่ม ➕ ด้านขวาบน กรอกชื่อทริป เลือกจุดแวะและจังหวัดที่วิ่งผ่าน พร้อมเลือกงบประมาณเบื้องต้น

💡 คีย์พิเศษ: กดปุ่ม "✨ คำนวณอัตโนมัติ" ระบบจะค้นหาพิกัด GPS เพื่อคาดคะเนระยะกิโลเมตรและเวลาล้อหมุนล่วงหน้าให้เองทันที!

2. การแก้ไขพิกัดหน้างานจริง (Timeline)

ระหว่างขับรถออกทริป คุณสามารถปรับพิกัดหน้างานได้อย่างอิสระบนแผนเดินทาง:

  • กด ✏️ แก้ไข เพื่อเปลี่ยนชื่อจุด, ลบจุด หรือปรับเปลี่ยนระยะทาง
  • กดปุ่ม ➕ แทรกจุดแวะเพิ่มตรงนี้ เพื่อเพิ่มร้านอาหาร/ปั๊มน้ำมันระหว่างทาง
  • หลังจากปรับแต่งเสร็จ แนะนำให้แตะปุ่ม 🤖 อัปเดตเส้นทาง ด้านบน เพื่อรีเซ็ตระยะทางขับรถและเวลาถึงใหม่ทันที

3. การลงประวัติเช็กอินและนำทาง

แตะปุ่ม 🗺️ นำทาง เพื่อเรียกเปิดแอป Google Maps นำทางนำร่องทันที และเมื่อขับรถไปถึงสถานที่แล้ว แตะปุ่ม "เช็กอินพิกัดนี้" เพื่อบันทึกประวัติความจำและลงบันทึกเวลาจริง (Actual Arrival)

4. บันทึกและวิเคราะห์ค่าใช้จ่าย (💸)

สามารถคัดกรองพิมพ์กรอกเงินจ่ายจริงใต้กล่องของสถานที่แต่ละแห่ง โดยเลือกจัดจำแนกเป็น 4 กลุ่มงบประมาณหลัก ระบบจะทำการบวกรวมและแยกประเภทวิเคราะห์ส่งไปจัดทำรายงานรวมหน้า "สรุปเงิน" ทันที

หากยกเลิกแผนไม่แวะจุดนี้แล้ว:

➕ แทรกจุดแวะเที่ยวระหว่างทาง

💡 คุณสามารถเว้นว่างระยะทางและเวลาออกไว้ได้ จากนั้นกด "เซฟแทรกจุด" แล้วกดยิงคำนวณระยะทาง GPS ใหม่ทั้งหมดที่ปุ่มบน Timeline ได้เลย!

➕ เพิ่มหมวดงบประมาณในทริปนี้

⚙️ การตั้งค่าเครื่องมือนำทาง

ระบบจะดึงระยะทางขับรถจริงและระยะเวลาเฉลี่ยผ่านระบบแผนที่แบบอัจฉริยะ

รหัส API Key ของคุณจะบันทึกไว้ใน Browser ของมือถือเครื่องนี้อย่างปลอดภัย เพื่อใช้ยิงดึง Distance Matrix

🔒 เพื่อความปลอดภัยขั้นสูงสุดในการใช้งานจริง: โปรดตั้งค่าข้อจำกัดการเข้าถึง (API Key Restrictions) บน Google Cloud Console ให้รับเฉพาะ HTTP Referrers โดเมนของคุณเพื่อป้องกันการถูกสวมรอยใช้คีย์จากที่อื่น
